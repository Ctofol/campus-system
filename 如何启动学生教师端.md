# 如何启动学生/教师端（登录注册页面）

当前项目里有两个前端，容易跑错：

| 目录 | 说明 | 启动后看到的页面 |
|------|------|------------------|
| **fronted** | 学生端 + 教师端（灵析运动 App / H5） | **登录、注册、首页、运动、课程** 等 |
| **admin/frontend** | 管理端 H5 | **管理后台**（班级管理、账号管理、批量导入） |

你截图里看到的「管理后台」说明当前运行的是 **admin/frontend**，不是学生/教师端。

---

## 在 HBuilderX 里正确启动学生/教师端

1. 在 HBuilderX 中**不要**以整个 `campus-system` 根目录作为运行项目。
2. **用 HBuilderX 打开 `fronted` 文件夹**：
   - 菜单：**文件 → 打开目录**
   - 选择：`campus-system/fronted`（不要选 campus-system 或 admin）
3. 在 HBuilderX 里确认当前项目是 **fronted**（左侧资源管理器顶部应显示「fronted」或「灵析运动」）。
4. 再点 **运行 → 运行到浏览器** 或 **运行到手机/模拟器**。

这样启动后，首屏会是 **entry 再跳到登录页**（未登录时），即学生/教师端的登录、注册页面。

## 微信小程序模拟器（避免 `app.json is not found`）

本项目是 **uni-app**，源码里是 `pages.json`，**根目录没有 `app.json`**。必须先编译，微信开发者工具才能启动。

### 推荐方式（自动打开正确目录）

1. 用 HBuilderX 打开 **`fronted`**（不要打开整个 `campus-system`）。
2. 菜单：**运行 → 运行到小程序模拟器 → 微信开发者工具**。
3. 首次会编译并生成：`fronted/unpackage/dist/dev/mp-weixin/`（内含 `app.json`），并自动用该目录打开微信开发者工具。

### 若仍报错或手动导入过项目

- **不要**把导入目录选成 `fronted` 或 `campus-system` 根目录。
- 应选编译输出目录（二选一）：
  - 开发调试：`fronted/unpackage/dist/dev/mp-weixin/`
  - 发行包：`fronted/unpackage/dist/build/mp-weixin/`
- 若 `dev/mp-weixin` 不存在或为空：先在 HBuilderX 执行一次「运行到微信开发者工具」再导入。

**注意：** 不要在 `fronted` 根目录放带 `miniprogramRoot` 的 `project.config.json`，否则编译进 `mp-weixin` 后微信工具会找错路径（报「在该目录下未找到 app.json」）。微信工具应直接打开 **`unpackage/dist/dev/mp-weixin`** 这一层，且其中的 `project.config.json` 不要写 `miniprogramRoot`。

---

## 简要对照

- 要看到 **学生/教师 登录、注册页** → 在 HBuilderX 里打开并运行 **`fronted`**。
- 要看到 **管理后台**（班级管理、账号管理等）→ 运行 **`admin/frontend`**（或用 `npm run dev`）。
