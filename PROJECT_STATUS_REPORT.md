## 🔧 技术实现

### 前端
- **框架**: Uni-app (Vue 3 Composition API)
- **状态管理**: Vue Reactive
- **网络请求**: uni.request 封装
- **地图服务**: 腾讯地图API
- **平台支持**: 微信小程序、H5、App

### 后端
- **框架**: FastAPI
- **ORM**: SQLAlchemy
- **认证**: JWT Token
- **数据库**: PostgreSQL / SQLite
- **文件上传**: 本地存储

### 核心功能实现
- **定位追踪**: GPS + 加速度传感器
- **步数统计**: 加速度传感器波峰检测
- **视频上传**: chooseMedia API
- **AI评分**: 后端算法评分
- **实时数据**: 定时轮询更新
