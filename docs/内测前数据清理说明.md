# 内测前数据清理说明

## 清理目标

用于学生端内测前重置业务痕迹，保留账号、班级、课程、任务等基础数据。

默认清理：

- 跑步、体测活动记录
- 活动指标、AI 分析结果、教师评分
- 健康申请
- 通知
- 课程报名和学习进度
- 跑团活动报名记录
- 学生已获得勋章记录
- 学生临时状态：健康状态恢复 normal、异常原因清空、常规成绩归零

默认保留：

- 学生账号、教师账号
- 学号、密码、班级、专业
- 教师和班级绑定
- 课程和课程内容
- 任务
- 跑团和跑团活动
- 勋章定义

## 使用方式

先预览，不会真正删除：

```bash
python backend/scripts/prepare_internal_test.py
```

确认后执行清理：

```bash
python backend/scripts/prepare_internal_test.py --yes
```

执行时如果当前数据库是 SQLite，脚本会自动在 `backups/` 目录生成一份备份。

## 可选项

同时删除任务：

```bash
python backend/scripts/prepare_internal_test.py --yes --delete-tasks
```

同时删除课程：

```bash
python backend/scripts/prepare_internal_test.py --yes --delete-courses
```

同时删除跑团：

```bash
python backend/scripts/prepare_internal_test.py --yes --delete-run-groups
```

不建议内测前默认使用这些可选项，除非你想重新创建任务、课程或跑团数据。
