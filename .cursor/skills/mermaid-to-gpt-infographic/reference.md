# 附录：提示词变体与对话骨架

主流程（Skill 3.1 → 本技能 → `fig-*.mmd` → [mermaid.live](https://mermaid.live) 导出 PNG → ChatGPT）以 **SKILL.md** 中的「标准流水线」与 **「Agent 交付清单」** 为准；本节为可选变体与对话技巧。

## 变体 1 — 强调「警校体测 / 翊晨运动」叙事

在「主提示词模板」的【内容结构】中追加（仅当用户业务确为此类时）：

```text
视频侧五阶段需可见：视频输入 → 姿态关键点 → 动作识别 → 次数与标准判断 → 评分输出。
跑步/GPS 侧注明：精度过滤、速度上限、距离门限、轨迹上传（若本次图不包含跑步可删）。
后端留痕：轨迹摘要、体测视频 evidence、审核/质检标记（可用小号字或脚注框）。
```

## 变体 2 — 只要线稿（便于 PPT 上色）

```text
只输出黑白线稿信息图，无填充色；线宽统一；所有标签用无衬线体大号占位字「L1」「L2」。
我将在 PowerPoint 中自行上色与替换中文。
```

## 变体 3 — 三张图统一视觉系统

```text
我将分三次请求三张图（流程图/时序/拓扑）。请第一次输出时定义色板与字体规则；后两次严格复用同一色板与箭头样式，仅改拓扑结构。
```

## ChatGPT 对话骨架（用户可复制）

1. 上传：`style-ref.png`（参考版式）、`structure-ref.png`（Mermaid 导出 PNG，可选）。  
2. 第一条消息：粘贴 `SKILL.md` 中的「主提示词模板」并填好 `{...}`。  
3. 第二条消息（若字错）：「请保持构图不变，仅把下列标签替换为：…」逐条列出。  
4. 第三条消息（若要素缺失）：「请在与第一张相同的版式下，增加并行泳道：…」  

## mmdc 批量导出（可选）

在含多个 `.mmd` 的目录中（需已安装 Node）：

```bash
for f in docs/diagrams/*.mmd; do npx @mermaid-js/mermaid-cli -i "$f" -o "docs/assets/$(basename "$f" .mmd).svg"; done
```

Windows PowerShell 可用 `Get-ChildItem` 循环，路径按用户仓库调整。
